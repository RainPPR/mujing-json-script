# Mujing JSON Script (幕境词典数据处理工具)

一个用于批量增强 **幕境 (Mujing)** 词典 JSON 文件的自动化工具。该工具通过调用有道词典 API，为词汇自动补充美式/英式音标及中文释义，并具备自适应并发控制功能，确保在大批量处理时的稳定性。

## 🌟 核心特性

- **自动化增强**：自动从有道 API 获取词汇的 `usphone`（美音）、`ukphone`（英音）以及多种中文释义。
- **自适应并发控制**：内置 `ConcurrencyManager`，根据 API 的响应状态实时调整并发线程数。遇到频率限制（403/429）时自动缩减并发并触发退避算法；运行稳定时自动恢复并发，最大化处理效率。
- **断点续传机制**：通过 `config.json` 维护处理状态，自动跳过已完成的文件和分类，适合长时间运行的大规模数据任务。
- **多线程加速**：使用 `ThreadPoolExecutor` 并行处理单个文件内的词汇请求。
- **健壮性优化**：包含指数退避重试逻辑，并在遇到持久性非频率错误时安全退出，防止数据损坏。
